Timestamp Microservice API

Congratulations! You've reached the root of this Timestamp Microservice API.

Usage

This API is super simple. It accepts with a UNIX timestamp (1450137600) or a natural language date (January 17, 1970).

Simply append either one to the end of the URL, and you'll get something like this: {"unix":1411200000,"natural":"January 17, 1970"} in return.

That's it!

Example Calls

Valid Calls:

The main rule to remember is the date passed in must be able to be correctly turned into a Javascript date object via new Date()

                        // Unix Timestamp
                        // returns {"unix":1450137600,"natural":"January 17, 1970"}
                        fetch('http://localhost:8000/1450137600')...
                    
                        // Natural Language Date (URL encoded)
                        // returns {"unix":1450137600,"natural":"January 17, 1970"}
                        fetch('http://localhost:8000/January%2017,%201970')...
                    
                        // Natural Language Date (with spaces)
                        // returns {"unix":1450137600,"natural":"January 17, 1970"}
                        fetch('http://localhost:8000/January 17, 1970')...

Invalid Calls

                        // Gibberish...
                        // Returns ^^
                        fetch('http://localhost:8000/adsfasdfadsf')...
                    
                        // Invalid Date String
                        // returns "Invalid date. Please enter a UNIX timestamp (ex 1450137600), or a natural language date (ex January 17, 1970)."
                        fetch('http://localhost:8000/January 17th, 1970')...
                    
                        // Natural Language Date with no spaces or encoding
                        // returns "Invalid date. Please enter a UNIX timestamp (ex 1450137600), or a natural language date (ex January 17, 1970)."
                        fetch('http://localhost:8000/January17th,1970')...
